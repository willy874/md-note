# 稀有前端工程師職缺釋放

我們公司要找前端工程師啦～～

有興趣私訊我你的履歷唷

必要技能：

- React, TypeScript

以下都是真實使用技術，沒有全部會也沒關係

- Framework: Vue2, React18, Micro Frontend
- Library: Antd5, zod, axios
- CSS: tailwin, styled-component, emotion
- Test: Jest, Vitest
- Builder: webpack, rollup, nx, pnpm workspace
- Docker

其他團隊使用工具

- Backend: Python(Fest API), Go, TypeScript
- SRE: K8S
- QA: Python(selenium)
- Team: Clickup, wiki, google doc, Slack, github

有以下經驗佳：

- 參與過 Micro Frontend 開發
- 有開發過 npm package

團隊優點：

- 完整的開發流程，planning, kick off, testing… 極少的隕石，讓你體驗什麼是軟體開發！
- 有前端小組會議，每週一起討論團隊制度與開發優化
- 沒有政治鬥爭，大家都努力完成工作
- Code Review 制度，沒有被審核的 Code 是不能合併的！
- 有前端共用的 package，你可以深度學習前端技術
- 扛得住的主管，老闆的隕石會幫你消化
- 有 Sanse 的 PM，有優質完善的 Spec 文件（資深 PM 還懂 git, docker……)
- 堅強的 QA，我們家的 QA 要寫自動化，不是那種手動測試
- 完善的 CI/CD ，什麼都自動化（還在努力進化
- 公司在台北車站，吃飯便宜，交通方便
- 開共用 API 要 Design Review，沒有什麼開好了自己看

團隊缺點：

- 目前“總平均”測試覆蓋率只有 35% 左右，努力提升中
- 技術門檻高，一開始很痛苦
- CTO 技術雖然超強，但不擅長溝通ＸＤＸＤ
- 開發流繁雜，步驟繁多
- 沒什麼獎金制度
- 沒有前端主管
- 沒有遠端

薪資：90w/y ~ 130w/y (初步估算，也許還能談)
