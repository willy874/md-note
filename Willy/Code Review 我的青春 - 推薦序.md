
**無關時空與時代，對於這個領域熱情依舊**

在台灣前端的網路社群蓬勃發展時，因緣際會與 Ray 在同一個社群團體認識，經歷大大小小有趣的活動，看到他對於工作、技術等等的熱情由衷佩服。

看他在六角學院各種活動與平台發光發熱的表現，真是一個默默但耀眼的人。他努力把每一個用心的學生照料好，看他在處理各種計畫、批改作業、學生輔導，各項事務無微不至。他不是高調表現的那種，而是默默守護在每個學員身邊，帶著學生成長茁壯。如果說六角學院是台灣前端工程師的搖籃，那 Ray 就是前端工程師圈內的媽祖了（？？？）。

有在教學的都知道，要能夠教好學生，必須花多時間精深每個知識點，對於每個知識點需要更加熟練精通掌握，才能夠清楚簡單的把一個複雜的知識說明清楚。在漫長教學歲月中，他精煉了自己的知識、經驗、文辭，產生這本豐富有趣的科普作品，把 30 年來計算機的發展歷史拆解成易懂的說明書，也真是他才能說清楚這種枯燥乏味的知識。

那段時光⋯⋯從數據機、映像管，一路到筆電手機。從電腦擁有者是特權的權杖，一路到人手都有的設備。從開著筆記本像施法一樣寫程式，一路到現在 AI 的世代動動嘴巴 App 就做出來。這些世代的流轉與變遷遷，看完不禁又有點感傷，原來這世界變化這麼大。小時候各種回憶湧上心頭，這個 Code Review 像是走了人生跑馬燈一樣的 Life Review，太過真實。

未來的人已經不在意過去發生什麼事情了，一切的技術都是如此理所當然，一切的工具都是理所應當。歷史的洪流過去了，這本書留下一些紀錄與足跡，滄海桑田之中，我們這些老人看了也可以會心一笑了！

—— 竹子｜前端工程師
