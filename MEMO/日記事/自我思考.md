# 自我思考

#

嘗試

與其去嘗試，她會選擇穩定，害怕踩雷。出外選擇店家心態尤其如此，沒有驚喜，沒有特殊，只有預期中的東西。

結婚與家人

自己面對家人催婚的壓力，不參與。

如果事先約到台北，可以見面，但不想跟我跑高雄。理由是花錢、貓等等因素。

穿著打扮

她覺得我會被怎樣，不要給她看到出入家門穿那樣，她覺得不安全。

但對於這些打扮，她不理解，也不參與。

關於自己，保持自己的習慣。

參與交友

她覺得她需要自己的空間，跟著我的交友太累了。她自己的交友她需要沉澱，先尋找自己的興趣，後面再推展。

下班後的談話

不理解，但可以先用訊息表達，離開家裡這個空間，會吵到她。

承諾

她不願做出改變的承諾，認為自己做不到而不做出承諾，也不願意接受自己講話帶給人壓力。

傷害

她擁有特權可以傷害人，無論是言語、咬、捏、打，而我從一開始就沒有這個權利。她可以用情趣包裝，但我卻無法理解。

刺激

總是會對我各種新鮮嘗試、特殊事件帶來言語刺激與訕笑，並沒有想接納，恥笑成分多。

傳統與前衛

在傳統觀念與前衛觀念的制高點自助餐，選擇自己喜歡的結果，而不接受各種傳統綁架，但也不願意面對新興觀念的挑戰。

親密關係

避開不談。
