# 產值

# 衡量

#

很多人都喜歡用「產值」來計算這個人的價值，然而，何謂產值？

工程師完成一個功能，產值多少？

客服解決一個客訴，產值多少？

設計師規劃一個功能，產值多少？

架構師規劃一個專案，產值多少？

排除市場上浮動不確定的報價，其實這些根本無法被衡量。最容易衡量的只有業務賣出的產品，但營業額最高的業務就是最有價值的嗎？

那帶著團隊沒有賣產品的業務主管呢？

做好客戶服務的業務呢？

整理好客戶文件的業務呢？

收集最多潛在名單的業務呢？

這些看似「沒有產值」的工作卻是幫助團隊很重要的齒輪，一昧追求產值只是一種膚淺的行為。

每個職位有每個職位可以追求的指標，而非靠產值來談價，因為一個產值背後的龐大成本和複雜度並非是一個人的成果，那是背後付出努力的每個人貢獻所帶來的結果。
