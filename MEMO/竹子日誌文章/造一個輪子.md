# 造一個

# 輪子

#

想要造一款輪子之前

首先你要了解別人的輪子解決了什麼問題？

了解別人的輪子使用上極限在哪？

最後發現有致命缺陷的問題，你決定選擇刻一個輪子。

這個思考脈絡和功課要做足。

如果沒辦法參透別人輪子的精髓，那代表你造輪子的功力還不成氣候，那你造出來的是個方輪，使用它只會帶來災難。

當你覺得「做不到」時，先思考一下是你不夠了解別人的工具？還是真的需求非主流？或是你正在開創一個別人沒想過的解決問題思路？
