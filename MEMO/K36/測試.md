目前先讓測試可以運作。

目前有幾個問題：

- 因為內部採用的是手刻的 xhr，跟 msw 的攔截功能有所違背，所以目前 api mock 先對 `ajax` 進行攔截。

- 因為 jest 和 webpack 是脫鉤的，如果有一些針對打包過程的加工邏輯，這都需要額外去在 setup file 去 mock 相關變數。

- 持續重構舊的有問題的測試
