# 人生哲學

1.虛懷若谷，幽默風趣

2.接地氣便能看到機會

3.需要的是可以依靠的心腹

4.有利則正義，不利變小人

5.把事情做好，賺錢是副作用

6.不要只為了完成，必須追求更好的品質

7.正確的鼓勵來管理

8.有時贏在氣勢就好

9.去看經營者做了什麼事

10.錢只是軟體，自己的腦袋才是硬體

11.有時，慢下來想一想，思考一下

12.生活就是出門雞叫，回來換鬼叫

13.前端思維，求新求變。後段思維，穩扎穩打。

14.我用心就不談殺價

15.橫跨產業，整合技術

16.因應當時時空背景找出解決方案

17.聊聊有趣的事，人就會有趣

18.美團外賣，團隊分工，以外借力

19.成為解決問題的提供者，本身便是品牌產品

20.用立體的規劃來作為行銷開闊

21.如何整合老經驗與新技術？

22.合作關係是需要合約的

23.書面資料是合夥價值的擔保

24.去觀察看不到的，看到的只是現象

25.企業危機？怎麼處理？

26.軟實力是透過觀察學習的

27.不要只看表層，多多觀察與學習內涵與門道

28.多問為什麼，問自己，問別人

29.兩年換一個產業，都要學習不同的角度

30.告訴對方大家都是做生意的，不要浪費時間在自己身上

31.領導不是單純的靠能力讓大家追隨，而是靠魅力讓大家聚在一起。

32.不要去抱怨和貶低，如何解決問題才是真的。建立一個讓朋友們覺得舒服的環境，讓大家聚在一起。

33.自己的感受從來都不重要，重要的是傾聽和達成什麼東西。

34.撰寫文字是最大的累積，每天一定要寫一些東西，程式也好，生活體悟也好，有所累積都是好的。

35.不要在意識不清晰時做重大決定
