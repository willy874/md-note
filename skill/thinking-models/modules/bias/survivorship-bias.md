# 倖存者偏誤（Survivorship Bias）

## 核心概念

倖存者偏誤：**我們看到的是留下來的樣本，看不到被淘汰的樣本，卻把倖存者的特徵當成成功的原因**。

經典故事：二戰期間，美軍分析返航轟炸機的彈孔分布，發現機翼、尾部中彈最多，考慮在那些位置加強裝甲。統計學家 Abraham Wald 指出：**這些是能飛回來的飛機**。真正該加強的是「機頭、引擎」——因為中彈在這些位置的飛機全部墜毀了，根本沒進樣本。

核心結構：

- 你的資料集 = 倖存者
- 失敗者 = 不見了 / 沒記錄 / 不發聲
- 你歸納的「成功因素」 = 其實是倖存者的**相關性**，不是**因果**

危險在於，失敗者不會出書、不會上 TED、不會被 YouTube 演算法推——你**物理上看不到他們**。結論永遠是基於半個樣本。

## 使用時機

當使用者處於以下情境，載入此 submodule：

- 使用者研究**成功者的做法**並想複製（「這些 CEO 的共同習慣」「獨角獸創業者的 5 個特質」）
- 讀**創業傳記、暢銷書、YouTube 經驗分享**後想如法炮製
- 評估**投資策略 / 基金績效**時，只看「還在榜上」的那些
- 看**學長姐成功案例**就覺得某條路容易走
- 聽到「我試了這個方法賺大錢」的故事就急著跟進
- 研究**歷史**時只看留下來的證據

不適合時機：

- **樣本沒有倖存者偏誤**：有完整全體資料（含失敗者），如某國全數上市公司、某醫院全部病例
- 在做**創作 / 欣賞**（欣賞傳奇樂團的音樂），和評估「我該怎麼成功」是兩件事

## 如何應用

1. **強迫問：「同樣做法失敗的人在哪？比例多少？」**
   - 這個 CEO 早起冥想 → 有多少早起冥想的 CEO 失敗了？（很多，你沒看到）
   - 這本書教的投資法 → 用這個方法賠錢的人在哪？（賠錢不會寫書）
2. **找基準率（base rate）**：
   - 創業五年存活率、新店關店率、訓練後的職業球員比例——這些數字才是你要對照的
3. **看整個「分母」，不只「分子」**：
   - 分子：倖存者
   - 分母：所有曾經嘗試的人
   - 成功率 = 分子 / 分母，不是「我看到很多分子所以機率高」
4. **追問倖存者的「運氣成分」**：多數倖存者的成功都是**技能 × 運氣**的乘積。沒有失敗樣本就無法估計運氣權重。問：「如果重跑 100 次，這個人還會贏嗎？」
5. **刻意研究失敗案例**：讀「死掉的公司」「破產投資人」「退出學術的人」。失敗的分布告訴你什麼會殺死你。
6. **小心「歸納式」內容**：書名是「X 成功者的 N 個特質」的幾乎都是倖存者偏誤的載體。不是說內容沒價值，但**相關 ≠ 因果 ≠ 可複製**。

## 案例

- **Wald 的飛機**：經典原型。沒看見的樣本（墜毀飛機）才是答案所在。
- **《從 A 到 A+》/ 《追求卓越》等管理暢銷書**：分析「卓越企業」的共通習慣。問題是——很多書中推崇的公司幾年後表現平庸甚至破產（例：柯林斯書中的部分公司）。當年的「卓越因素」其實是當時環境 + 運氣。
- **創業家的作息 / 習慣**：很多人說「成功 CEO 都早起 / 冷水澡 / 讀很多書」。但同樣早起冷水澡讀書的失敗創業者可能有幾萬個，只是他們沒有出書機會。這些習慣可能有益健康，但**不是成功的充分條件**。
- **基金績效**：基金排行榜上常看到的「五年年化 20%」——是因為**表現差的基金被關掉或合併**，榜單永遠是倖存下的那批。真實的平均報酬遠低於榜上的數字。
- **演藝圈 / 體育圈**：「你看 XX 沒念書也紅了」——但沒念書也沒紅的有百萬人。倖存者偏誤會讓人低估「穩定職涯」的長期價值。
- **YouTube / 社群成功學**：教你怎麼做內容一夜爆紅的頻道，本身就是爆紅樣本。沒爆紅的幾千個同類頻道沒機會出現在你的推薦頁。
- **房地產 / 股票長期報酬**：「台股過去 50 年年化報酬 X%」——這個數字是現在還存在的股市的表現。幾十個 20 世紀早期的股市（俄國、阿根廷、奧匈帝國）崩潰或清零了，但它們不在統計裡。全球真實的股市長期報酬遠低於倖存國家的報酬。

## 常見誤用

- **以為「避開倖存者偏誤就等於否定成功學」**：不是。倖存者的故事仍有資訊，只是要加上「失敗者的分布」才完整。
- **過度悲觀癱瘓**：發現倖存者偏誤後覺得「那成功都是運氣，我也努力沒用」——另一種極端。正解是校準期望 + 選**分母比較友善**的賽道。
- **把倖存者偏誤當懶惰的擋箭牌**：別人提起成功案例就說「倖存者偏誤」草草否定，沒真的去找失敗分布。工具是研究，不是嘴砲。
- **忽略「可複製度」差異**：有些因素就算在失敗者身上也普遍（努力、基本功），這種仍值得複製；有些只有成功者才有（運氣、時代紅利），這才是不該照抄的。
- **用一個反例否定整個模式**：「這個方法 XXX 失敗了」不等於該方法無效。要看**整體分布**，不是單一反例。

## 與其他模型的關係

- **+ 確認偏誤**：搜集倖存者故事 + 從這些故事裡找支持自己信念的教訓 = 雙倍扭曲。
- **+ 可得性偏誤**：倖存者有故事、有曝光、容易被記得 → 在你腦中的樣本嚴重偏向成功者。
- **+ 敘事謬誤**：倖存者的人生被包裝成「奮鬥 → 關鍵選擇 → 成功」的故事，但失敗者的人生也有同樣結構，只是沒被寫出。
- **+ 第一性原理**（analysis 模組）：不看倖存者清單，而是從「這個活動的物理 / 經濟限制」推演成功的條件機率。
- **+ compounding / leverage**（action 模組）：評估任何「槓桿賽道」（媒體、SaaS、投資）時要用分母視角——多數人敗在哪裡比少數人贏在哪裡更有指導價值。
- **+ decision 模組的期望值**：期望值必須用全樣本（含失敗）才有意義。只用倖存者算 → 期望值會被嚴重高估。
