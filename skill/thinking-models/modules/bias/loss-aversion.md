# 損失規避（Loss Aversion）

## 核心概念

損失規避由 Kahneman 和 Tversky 在「前景理論」（Prospect Theory, 1979）中提出，這篇論文讓 Kahneman 拿到諾貝爾經濟學獎。核心發現：**失去一件東西的痛苦，大約是獲得同等價值東西快樂的 2 倍**。

具體實驗：
- 給你 $100 的機會 vs 失去 $100 的風險——多數人拒絕公平賭局（50/50 贏或輸 $100）
- 要讓人願意參與 50% 輸 $100 的賭局，要配上 50% 贏 **$200 以上**
- 這個「兩倍」的比例稱為「損失係數」，後續研究估計在 1.5–2.5 之間

損失規避的副產品：

- **稟賦效應（Endowment Effect）**：一旦某物屬於你，你會要求更高的價格才願意放手
- **現狀偏誤（Status Quo Bias）**：傾向保持現狀——改變有「失去現狀」的痛
- **沉沒成本謬誤**：不願停損，因為承認損失比繼續投入更痛
- **處置效應（Disposition Effect）**：投資人太早賣賺錢的股、太晚賣賠錢的股

核心洞見：**損失規避本身不是錯，是演化遺產**——祖先面對「失去食物」的代價遠高於「多拿一份」的好處（後者邊際效用低、前者可能致命）。在現代決策中，這個權重常常失調——多數「損失」其實是可承受的，但我們照樣用古代的情緒反應。

## 使用時機

當使用者處於以下情境，載入此 submodule：

- 使用者**怕失去現狀**而錯過明顯更好的機會（不換工作、不停損、不離開壞關係、不搬家）
- **沉沒成本**推著走：「已經投入這麼多了」（時間、金錢、感情、學歷）
- 股票 / 投資**不敢停損**，繼續加碼攤平
- 面對「穩定小得 vs 不穩定大得」選項時，總選前者；同時面對「穩定小賠 vs 不穩定大賠」時，反而賭大賠
- **新 vs 舊**決策：舊方案的缺點看不見（已適應），新方案的風險被放大
- 團隊決策中反對改變的聲音不成比例地大

不適合時機：

- 所謂「損失」**真的不可承受**（真的會破產、身體毀損）——那不是偏誤，是合理風險管理
- 使用者的保守是**長期最優策略**（例如退休金的資產配置）——不用硬戳

## 如何應用

1. **重新框架：不是「失去 X」，而是「用 X 交換 Y」**：
   - 換工作 = 失去現職 → **用現職交換 Y 機會**
   - 停損 = 失去已投入 → **用剩餘資金換未來的自由度**
   - 分手 = 失去關係 → **用這段時間換健康的下一段**
2. **問「如果我現在沒有這個，我會主動去取得嗎？」**：這個問題是稟賦效應的解毒劑。
   - 如果答案是「不會」，那你就是因為「已經擁有」才捨不得，不是因為它真的有價值
3. **用期望值計算，不用情緒**：
   - 不要問「我會痛嗎？」
   - 問「在 100 個平行宇宙裡，這個決定的平均結果是什麼？」
4. **意識到 2:1 的損失係數，主動校正**：
   - 感覺「微虧」的機會 → 實際上可能是「微賺」
   - 感覺「該放棄」→ 可能只是痛被放大
   - 降權你的痛感約 50%，再評估
5. **沉沒成本歸零**：
   - 決策只看**未來**的期望值，不看過去的投入
   - 「如果我從今天才接觸這件事，我還會選擇投入嗎？」是標準拷問
6. **切成小步**（action 模組 MVP / 兩分鐘法則）：大改變觸發損失恐懼最強。切小 → 每一步失去的都小 → 容易啟動。
7. **記住對稱性**：你拒絕的「失去 $100」的機會，可能正是別人「獲得 $100」的機會。你的恐懼未必對應真實風險。

## 案例

- **不敢停損**：股票從 100 跌到 70。「賣了就實現虧損了」——但沒賣也是虧損，只是帳面上。問：如果現在手上有現金，你會用 70 買這支嗎？不會？那就該賣。
- **套牢基金**：持有一檔表現差的基金多年，換掉要承認過去選錯。繼續持有則忽視了機會成本。每一年維持現狀都是一次主動「選擇不換」的決策。
- **不換工作**：現職不滿意但熟悉。新工作不確定性高。損失規避讓你嚴重高估「離開現職的損失」、嚴重低估「留下的長期成本」。
- **壞關係**：投入五年，分手等於「白費」。但不分只是繼續耗——真正的問題是「未來五年我想怎麼過」，不是「過去五年」。沉沒成本 × 損失規避的雙重陷阱。
- **學歷 / 轉職**：念了三年的博士但發現不適合。很多人撐完主要是因為「半途而廢 = 承認損失」。但經濟學上，已投入的時間已經沒了，問的應該是「未來兩年，這樣繼續 vs 換跑道，哪個期望值高？」
- **稟賦效應的實驗**：給一組學生一個馬克杯，問他們最低賣多少；給另一組學生同樣的馬克杯但是用「買」的，問最高願意付多少。賣價中位數 $7，買價中位數 $3。**同一個物品，擁有後價值翻倍**。
- **政策 / 組織改變**：改 KPI、改 org chart、改報帳流程時，有人一定會大聲反對——多數反對者不是因為新方案差，是因為失去「我熟悉的現狀」。領導者要預期這個比例。

## 常見誤用

- **什麼都貼「損失規避」標籤**：有些保守是合理的風險管理（退休資金、小孩教育費）。不是所有不敢換就是偏誤。
- **過度反損失規避變成魯莽**：發現自己有損失規避後刻意「反其道」，結果跳進高風險局。正解是**校準**，不是反向極端。
- **強迫別人「克服損失規避」**：別人覺得痛是真實的感受。你說「那只是損失規避」只是用模型傷人。框架自己用，不要拿來說教。
- **忽略真實的不可承受損失**：破產、身體傷害、家庭破裂——這些不是偏誤，是真實 downside。期望值計算要納入 downside 的嚴重度，不是無腦取平均。
- **把「稟賦」等於「價值」**：有時你擁有的東西**真的對你更有價值**（因為你有特殊用途、有情感）——不是全部都該用市場價判斷。

## 與其他模型的關係

- **+ 錨定**：買入價 = 錨。跌破錨就觸發損失感。兩者共同造成「套牢心理」。
- **+ 沉沒成本謬誤**（decision 模組）：沉沒成本就是損失規避在時間維度上的表現——已經投入的無法收回，但損失規避讓它繼續影響決策。
- **+ 現狀偏誤**：損失規避的兄弟姊妹——對「改變」的恐懼很大程度上源於「怕失去現狀」。
- **+ 可得性偏誤**：損失的痛很鮮明（記得久），獲得的喜悅較淡（記不久）→ 腦中樣本偏向「損失經驗」→ 強化損失規避。
- **+ decision 模組的期望值**：期望值強迫你用客觀機率計算，抵消損失規避的情緒放大。
- **+ action 模組 MVP / Timeboxing**：把大決策切成小實驗，每一步的損失都可承受，整體下來敢做的事變多。
- **+ compounding**（action 模組）：長期複利的最大敵人是「歸零事件」；損失規避在這裡是**有用的校準**（某些 downside 確實該避）。工具的價值依情境而定。
