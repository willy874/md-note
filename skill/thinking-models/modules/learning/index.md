# 五、學習與溝通

> 載入時機見 `SKILL.md` 的路由表。本檔為此類別的**子路由**——先對照下表判斷使用者的情境，再 Read 對應的 submodule 深入取用。

用來加速吸收知識，並把自己的想法有效傳達給他人。七個 submodule 對應七種不同的學習 / 溝通情境，不要一次載入全部。

---

## Submodule 子路由表

### 1. 費曼技巧（Feynman Technique） → `learning/feynman.md`

**何時載入**：
- 使用者**學了一個概念但講不清楚**、自己覺得「大概懂了」卻說不出口
- 要寫技術文章、做教學、準備演講、答辯，需要把複雜概念講給非專業聽眾
- 「學過又忘」是常態——多半根本沒真懂，只是熟悉感（fluency）冒充理解

聚焦：**能教出來才是真懂**。用「教給 12 歲小孩」的極端白話檢驗理解漏洞。

---

### 2. 由上而下溝通（Pyramid Principle） → `learning/pyramid.md`

**何時載入**：
- 使用者寫完**被問「所以結論是？」**——鋪陳太久、重點埋在尾巴
- 寫給**時間很少的讀者**（老闆、客戶、跨部門主管）
- 郵件、週報、提案、投影片被嫌「太長看不完」

聚焦：**結論先行，理由次之,細節最後**。來源:麥肯錫 Barbara Minto。

---

### 3. 刻意練習(Deliberate Practice) → `learning/deliberate-practice.md`

**何時載入**：
- 使用者**練很久但卡在同一水平**（高原期 / plateau）
- 想把技能從「會」升到「厲害」：樂器、程式、寫作、運動、外語、設計、銷售
- 每次練都很舒服、沒出錯——這正是沒進步的警訊

聚焦：**在能力邊緣設計練習 + 即時回饋**。Anders Ericsson 研究。

---

### 4. 最小可學單位（Atomic Concepts） → `learning/atomic-concepts.md`

**何時載入**：
- 使用者**一次吞太多**、讀完整章還是混亂、筆記寫一堆回頭看不懂
- 學新語言 / 新框架 / 新領域：每個字都懂、整句不懂
- 想建立**長期可累積**的知識庫，而非上完就忘的課程進度條

聚焦：**把知識拆到最小獨立單元，逐一理解再連結**。搭配 Zettelkasten / Anki 最有效。

---

### 5. 類比思考（Analogical Thinking） → `learning/analogy.md`

**何時載入**：
- 使用者**跨入新領域**、現有知識看似無關，不知從何下手
- 要**教非專業人**理解技術概念（或反向翻譯）
- **創意發想**、命名、產品定位——需要借用熟悉結構

聚焦：**用熟悉領域的結構映射陌生領域**。小心:類比是入門工具,不是推論工具——要誠實標出失效的地方。

---

### 6. Steelman（鋼人論證） → `learning/steelman.md`

**何時載入**：
- 使用者和對方意見不同，**把對方翻譯成弱版本再反駁**（稻草人）
- 跨部門衝突、辯論、產品方向分歧、家庭 / 伴侶討論
- 讀到某個立場「覺得根本沒道理」——這正是該 Steelman 的警訊

聚焦：**先把對方觀點講到比他自己還強,再回應**。智識誠實的門檻,也是真正改變對方的前提。

---

### 7. 受眾思維（Audience First） → `learning/audience-first.md`

**何時載入**：
- 使用者寫完東西**讀者無感**、沒人轉、沒人行動
- 技術度錯位（對老闆寫工程細節、對工程師寫公關話術）
- 簡報 / 郵件 / 貼文 / 提案收不到預期回應

聚焦：**先問「讀者是誰、他在意什麼、他要做什麼」,再動筆**。所有溝通場景皆適用。

---

## 挑選提示

- **自己沒真懂** → 費曼技巧
- **結論沒浮出來** → Pyramid Principle
- **練了沒進步** → 刻意練習
- **學得太雜** → 最小可學單位
- **跨領域 / 教非專業人** → 類比思考
- **想說服 / 討論卡住** → Steelman
- **寫完沒人理** → 受眾思維

多數場景只對應一個 submodule。真的需要跨 submodule 時，典型組合：

- **學一門新技術**：Atomic Concepts（拆解）→ 費曼（驗證理解）→ 類比（串接已知）→ 刻意練習（熟練）
- **寫一篇說服文**：受眾思維（誰會讀）→ Pyramid（結論先行）→ Steelman（處理反對意見）
- **教學 / 簡報**：受眾思維 + 類比 + 費曼（三件套）
