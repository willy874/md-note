# 從眾效應（Bandwagon Effect）

## 核心概念

從眾效應：**當越多人相信或採用某事，個體就越容易跟著相信或採用，與證據強度無關**。「bandwagon」原指政治遊行中的大花車——跳上花車等於加入熱潮。

心理根源有三層：

1. **資訊性從眾**：當自己不確定時，「多數人的判斷」確實是有用的資訊來源（群體智慧有時為真）。
2. **規範性從眾**：不想被排除在群體之外——人是社交動物，意見孤立的生理不適是真實的。
3. **省認知**：每件事都獨立思考太耗能，跟著跑是大腦的預設。

Solomon Asch 的從眾實驗（1951）：在顯而易見的視覺判斷題中，如果小組裡其他人都故意給錯誤答案，約 75% 的受試者至少有一次跟著說錯。明知道答案是什麼，但群體壓力足以扭曲發言。

核心洞見：**「有多少人相信」和「有多少證據支持」是兩個完全不同的維度**。多數時候它們正相關（群體智慧），但在熱潮、泡沫、恐慌、政治極化中會嚴重脫鉤——這時從眾就是失準。

## 使用時機

當使用者處於以下情境，載入此 submodule：

- 「大家都這樣說」「這個超紅」是使用者的**主要理由**，而非一個輔助訊號
- **投資熱潮、meme stock、AI 熱、加密貨幣**——很多人衝進去只因為看到別人衝
- **網紅 / KOL 推薦**就覺得該買、該信
- 會議或社群中不敢提**異見**，因為多數人看起來同意
- **恐慌性拋售、囤貨搶購**（衛生紙、食物）——從眾的負面版
- 對某人 / 某立場的評價建立在「大家都說他 X」而沒有親自驗證

不適合時機：

- 某個共識**確實建立在長期證據**之上（例：疫苗科學、基礎物理）——不是每次從眾都錯
- 處於**實務工作需要快速對齊**的場合（專案執行、軍事行動）——效率大於獨立判斷

## 如何應用

1. **分開兩個問題**：
   - Q1：有多少人相信這件事？
   - Q2：有多少**獨立證據**支持這件事？
   - 理想是 Q2 先答完再看 Q1。Q1 在 Q2 之前看容易錨定。
2. **警覺「共識速度」**：一個觀點**突然從少數變多數**（幾週內）時特別可疑——通常是熱潮而非真實。真正的群體智慧通常累積緩慢。
3. **看主張的來源結構**：
   - 多數人是**各自獨立**得出結論？→ 群體智慧價值較高
   - 多數人是**互相抄、從同一兩個來源**擴散？→ 從眾放大，不是獨立信號
4. **問「如果我是第一個接觸這個議題的人，我會怎麼想？」**：強迫自己抽離群體壓力。
5. **刻意接觸異見**：不是為了反對，是為了讓自己能區分「共識是對的」還是「只是大家在跟風」。
6. **記錄自己當下的主張屬於哪類**：是我**思考出來的**，還是**看到很多人這麼說所以我也這麼說**？
7. **在群體中練習先表態**：Asch 實驗顯示，**只要有一個人先說不同意見，從眾率就大幅下降**。你若能當那個先開口的人，常為整個群體解鎖。

## 案例

- **Asch 實驗**：三條線段比長度，答案顯而易見。但當旁邊的「椿腳」全部故意答錯，約 75% 的受試者至少跟風一次，有一部分甚至每次都跟。這個結果被後續數百次重複驗證。
- **網路泡沫（1999–2000）**：「.com」公司估值脫離現金流，每個人都相信「這次不一樣」。沒參與的顧問、分析師受到強大同儕壓力被笑是「跟不上時代」——從眾推著泡沫變大。
- **Meme stock / crypto 狂熱**：GameStop、Dogecoin 等。多數後期進場的人不是分析過基本面，而是「看大家賺我也想賺」。泡沫後多數散戶虧損。
- **搶購衛生紙 / 泡麵**（疫情初期）：沒人短缺，但看到別人搶 → 自己也搶 → 真的短缺。從眾自我實現。
- **職涯選擇**：某段時期大家都去當 iOS 工程師；某段時期都去做 data science；某段時期都去做 AI。隨大勢進場的人多半會在熱潮退時發現競爭過度、待遇回落。
- **公司會議**：CEO 發言後沒人敢異議，決策變單邊。好的領導會**最後才發言**或刻意要求反對意見（Bezos 的 "disagree and commit"、Amazon 的 "silent start"）。
- **社會風氣 / 政治觀點**：社群演算法造成的同溫層——你所在的群組越同質，你越覺得「大家都這樣想」。其實只是你的訊息環境被過濾了。
- **科學史反例**：大陸漂移（Wegener）當年被地質學共識否定幾十年。板塊構造學說確立後才被接受。共識不是真理的保證。

## 常見誤用

- **變成無腦反從眾**：發現從眾後就刻意和多數唱反調——那只是換一種不思考。**獨立判斷 ≠ 永遠反對**。
- **把所有共識都打成「從眾」**：科學共識、基本事實（地球是圓的、疫苗有效）不是從眾，是長期獨立證據累積。過度廣泛打擊共識 → 滑入陰謀論。
- **只在「輸的一方」指責從眾**：自己的群體贏的時候不叫從眾，對手贏的時候才叫從眾——這是雙重標準。
- **用異見當耍帥**：為反對而反對，在每個場合都要當「唯一清醒的人」。這是社交表演，不是獨立思考。真正的獨立思考者會同意多數、偶爾反對，依證據而定。
- **低估群體智慧的價值**：在資訊不對稱、自己不懂的領域，**跟隨共識反而是理性的**（例：不懂醫學就聽醫生共識、不懂氣候就聽科學共識）。不是每次從眾都錯。

## 與其他模型的關係

- **+ [可得性偏誤](availability-heuristic.md)**：大家在談的議題變得可得 → 你覺得它重要 → 你也加入談 → 循環。
- **+ [確認偏誤](confirmation-bias.md)**：跟進共識後，只去找支持共識的證據。從眾 → 確認偏誤 → 更堅定。
- **+ [倖存者偏誤](survivorship-bias.md)**：熱潮的「成功案例」被放大（meme stock 賺大錢的人），失敗者被隱形 → 從眾者低估風險。
- **+ [敘事謬誤](narrative-fallacy.md)**：熱潮通常附一個動人故事（「這次不一樣」「顛覆產業」），故事讓從眾更容易發生。
- **+ [decision 模組](../decision/index.md)的[二階思考](../decision/second-order-thinking.md)**：「如果我跟進，後續會怎樣？如果所有人都像我這樣想，市場會變什麼樣？」——這是對抗從眾的有力工具。
- **+ [第一性原理](../analysis/first-principles.md)**（[analysis 模組](../analysis/index.md)）：不問「大家怎麼看」，問「底層事實是什麼」，然後重新推。從眾的解毒劑。
- **vs [prioritization 模組](../prioritization/index.md)的 [80/20](../prioritization/pareto.md)**：跟大多數還是跟少數高價值？常常答案是少數——但心理上很難不往多數靠。
