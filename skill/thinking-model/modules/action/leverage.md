# 槓桿思考（Leverage）

## 核心概念

槓桿的物理定義是「以小力撬動大物」。在職涯與商業情境中，槓桿指的是**讓一單位的自己，產生多倍的產出**。

Naval Ravikant 把現代槓桿分成四類：

1. **勞力（Labor）**：用他人的時間放大自己的產出。有人工作才能運作；管理成本高、ego 成本高。**古典槓桿**。
2. **資本（Capital）**：用錢放大產出。錢會睡，但會利滾利。**需要先有一桶金、需要決策力**。
3. **程式碼（Code）**：寫一次執行無數次。**零邊際成本**，24 小時運作，不用管理。
4. **媒體 / 內容（Media）**：文章、影片、podcast 一次創作，觸及無限人次。**零邊際成本 + 零 permission**。

前兩種槓桿需要「別人的許可」——別人願意替你工作、別人願意把錢借你。後兩種是**無許可槓桿（permissionless）**——不需要任何人點頭，網路允許你直接把東西放出去。Naval 主張這是這個時代最關鍵的紅利。

## 使用時機

當使用者處於以下情境，載入此 submodule：

- 覺得自己「每天很忙但影響力不大」、產出受限於親自動手的時間
- 想提升**收入天花板**、跳出「時間換錢」的線性模式
- 職涯規劃：考慮要走哪條路才有**規模化**的可能
- 商業模式設計：在思考 solo / 團隊 / 外包 的組合
- 副業選擇：猶豫是要接案、做 SaaS、還是寫內容

不適合時機：
- 處在需要深度練習專業（craft）的階段——過早槓桿會讓基本功不扎實
- 還不知道要做什麼的時候——槓桿是「方向確定後」的放大器，方向錯槓桿只會加速走錯

## 如何應用

1. **盤點現有槓桿**：你現在的產出靠什麼放大？純勞力？有員工？有程式？有受眾？
2. **評估每種槓桿的成本 / 報酬**：
   - 勞力：招聘、培訓、管理；可處理複雜事務
   - 資本：需要本金、需要選擇判斷；報酬天花板高
   - 程式：需要技術、需要維運；邊際成本近乎零
   - 媒體：需要時間累積受眾；一旦有觀眾則複利
3. **選擇最適合自己的組合**：
   - 內向、技術強：程式 + 媒體
   - 擅長交際、有資源：勞力 + 資本
   - 新鮮人 / 起步：先從媒體開始累積，零成本 + 練表達
4. **組合運用**：真正強的人通常多槓桿疊加。寫作家（媒體）→ 線上課（媒體 + 程式）→ SaaS（程式）→ 用利潤投資（資本）。
5. **避免過早加槓桿**：如果核心判斷力還沒練出來，加了槓桿反而放大錯誤。先 craft，再 leverage。

## 案例

- **SaaS 一人公司**：Pieter Levels 一人經營 Nomad List、Remote OK，年收七位數美金，純靠程式 + 媒體兩種無許可槓桿。
- **創作者經濟**：YouTuber 一支影片發布後觸及百萬人，這是媒體槓桿。開線上課等於把內容變成可重複販售資產。
- **程式工程師 vs 顧問**：顧問是勞力槓桿（用別人 + 時間換錢），工程師寫 SaaS 是程式槓桿（寫一次賺無限次）。長期天花板差距顯著。
- **投資人**：用資本槓桿，讓錢自己替你工作；代價是需要極強的判斷力，否則槓桿會反噬。

## 常見誤用

- **為槓桿而槓桿**：還在學寫程式就急著蓋 SaaS、還不會寫作就急著開電子報。**核心能力未到位時的槓桿會加速失敗**。
- **忽略槓桿的暗面**：
  - 勞力 → 管理與政治成本
  - 資本 → 判斷錯會破產
  - 程式 → 維運成本
  - 媒體 → 公眾形象一旦翻車傷害大
- **誤以為所有事都要規模化**：有些工作本質上就是「craft」——心理諮商、精品手作、頂級教練，其價值在於**不可複製**。硬套槓桿只會摧毀價值。
- **只看別人的結果**：看到 Pieter Levels 一年賺幾百萬，忽略他在那之前寫了很多失敗的 app、經營了十年的受眾。複利 + 倖存者偏誤。

## 與其他模型的關係

- **+ [複利思考](compounding.md)**：媒體、程式、資本 都是複利型槓桿。兩者疊加才是真正的飛輪。
- **vs [MVP](mvp.md)**：MVP 驗證方向、leverage 放大方向。順序很重要——先 MVP 確認市場，再加槓桿放大。
- **+ [80/20](../prioritization/pareto.md)**（[prioritization 模組](../prioritization/index.md)）：找到那 20% 最有槓桿效果的活動。
- **+ [bias 模組](../bias/index.md)**：要警覺[倖存者偏誤](../bias/survivorship-bias.md)——別只看槓桿成功者，大部分人敗在中途。
