# 團隊

# 不是

# 你想怎樣

# 就怎樣

#

在軟體的世界常常講到技術為王，但真的很強大家就會聽你的？坦白說未必。團隊是講究和諧、合作、統一，即使是你認為的最佳實踐，也未必可以實施於團隊。

曾經有人問我：「這個專案是你主導的，為什麼不OOO這麼做？」

坦白說，在團隊中，所有的決策並不是你說了算，多數還是需要有上層、夥伴等等因素去做技術決策，可能是權力大的說了算，可能是多數決，但主導不代表擁有絕對的實權，溝通協調取得共識的解決方案才是團隊。

對於某些技術高強的英雄類型工程師他們不理解這件事，他們認為你技術厲害當然決策出來的結果應該是最佳實踐。然而團隊有時候決策是取決於「最低水平者」或是「平均值」，還真不是想怎麼做就怎麼做。
