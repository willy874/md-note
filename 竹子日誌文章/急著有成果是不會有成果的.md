# 急著

# 有成果

# 是

# 不會

# 有

# 成果

# 的

Side project 對我來講不是一個為求職而準備的東西，所以我不會因為我想找工作才開始準備。

很多人都覺得 side project 是一個「我為了求職要準備的專案」，或是「我希望可以創造商業潛力」的專案。但很實際的是，大部分這些都不是吸引我做 side project 的理由。我要的是，我能夠在這個專案中玩到、學到、體驗到什麼。

我會踏實做好我覺得有價值的事情，一點一滴把每個細節都做好。
