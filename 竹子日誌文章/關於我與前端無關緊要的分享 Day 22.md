# 關於我與前端無關緊要的分享Day 22

# Build in Public

從頭去研究色彩系統，才發現現在 CSS 系統飛躍性的進步，跟過去顏色系統已經完全不一樣了。包含 color-mix, oklch 等等功能。

傳統要用基礎色延伸成透明疊色語法上滿限制，也不太能疊 CSS 變數混搭半透明。但有了 color-mix 功能反而這件事可行了。

覺得也是因為這樣，基礎色的使用更自由了，我決定用 mixin半透明來達到 color token 的變化。透過按比例疊色與疊色和半透明化，可以某種程度展示出特效感。

但我也發現致命的缺點，就是不能 transition，算出來的顏色不是一個容易得到結果的色彩，還需要搭配工具去確認最終結果。搞不好我找時間還得去改顏色系統的設定，不過就先做一個大概了。
