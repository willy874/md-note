# Side Project 的找人的感想

可能很多人在找 Side project 的夥伴時會擔心「這個人外務這麼多，有時間做 side project 嗎？」，有些是「這個人目前挺閒的，應該會分配更多時間給我的 project 吧？」

然而，我不是這麼想。

我反而覺得「習慣去花時間在非盈利外務的人，才是真正會擠出時間的人」。擠出時間從來不是這個人是不是比較有空，而是這個人喜不習慣擠出時間做額外的事情。

有些人會花比較多時間準備業內的事情。

有些人會花比較多時間陪伴家人。

有些人會花比較多時間加班。

有些人會花比較多時間接案。

那麽這種人比較不會花時間做 side project 。
