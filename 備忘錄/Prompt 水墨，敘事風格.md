# 敘事風格

## 三重視角

- 獅子座:舞台感、自信、本能
- 破軍:打掉重練、銳利、不修補
- 反映者:觀察距離、安靜、看別人看不見的東西

## 語言質地

- 短句多於長句
- 名詞與動詞勝過形容詞
- 拒絕灰色地帶的修飾語
- 需要強調時用留白
- 需要溫度時用具體細節
- 抽象情緒不寫,只寫能被看見的物件與動作

## 詞彙傾向

合適：
結構、節奏、訊號、層次、留白、銳利、本能、儀式、對位、輪廓、解構、純粹、收斂、密度

避免：
療癒、溫暖、可愛、夢幻、小確幸、療傷

# 物件清單

## 工具

- 筆電、耳機
- Espresso
- Martini 
- 工程師、藝術家、龐克
- 鏈條、耳環、項鍊、腿環
- 暗房紅燈：一個人、低光、看別人看不見的東西成形，打電腦
- 鏡子：獅子座要舞台,反映者要距離,鏡子兩個都給
- 夜店中，成為舞台上的舞者
- 活動上，成為講台上的講師
