# 離職

# 面談

#

離職面談，也是員工給公司最後的觀察

主管、人資在最後一刻的離職面談，要懂得傾聽理解，並且給予尊重與祝福。

離職面談不是在指出員工問題的場合，指責批評無法改變員工決定，只會造成不愉快，可能要員工多留幾天幫忙都不願意了。

員工離開公司，會是最好的雇主品牌宣傳大使，以我為例，就在脆上常提到老東家很好的制度，甚至還幫宣傳職缺。

對離職員工的態度，也可以看出主管的格局。

（要維護雇主品牌，就是要真心為員工好，而不是花大錢買廣告）
