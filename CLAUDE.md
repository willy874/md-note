This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

This is a personal knowledge base (Obsidian vault) containing markdown notes in Traditional Chinese. It is **not** a software project — there are no build steps, tests, or linting.

## Structure

- **竹子日誌文章/** — Blog articles (largest collection, ~400 files)
- **學習筆記/** — Technical learning notes (SQL, Docker, Git, Linux, System Design, etc.)
- **工作與專案/** — Work & project notes organized by company/project (H7, K36, PonderTech, Pentium, Travel AI, Frontend Infra, etc.)
- **Interview/** — Interview prep materials (questions, self-intro, CV, company-specific notes)
- **好文記錄/** — Curated article summaries and takeaways
- **備忘錄/** — Personal memos and life philosophy
- **社群貼文備案/** — Social media post drafts
- **歷史雜文/** — Historical/miscellaneous essays
- **英文練習/** — English practice
- **日記事/** — Personal diary entries
- **商業化/** — Business ideas
- **Willy/** — Personal brand content
- **skills/** — Claude Code custom skill definitions

## Conventions

- All content is in Markdown, primarily written in Traditional Chinese
- This is an Obsidian vault (`.obsidian/` config exists but is gitignored)
- Sensitive content lives in `.secret/` (gitignored)
- When creating or editing notes, match the existing style: Traditional Chinese, markdown formatting, no front-matter unless the file already uses it
